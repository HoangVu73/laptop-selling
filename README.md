# **Laptop Selling - Spring Boot**

## **Công Nghệ Sử Dụng**

### **Frontend**
- **HTML/CSS/JS**  
- **Bootstrap**

### **Backend**
- **Spring Boot**

### **Thư Viện & Framework**
- Spring JPA  
- Spring Security  
- Spring Web  
- Thymeleaf  
- Lombok  
- Validation

## **Yêu Cầu Cài Đặt**

1. Cài đặt **JDK** phiên bản 8 hoặc mới hơn  
2. Cài đặt **Maven**  
3. Cài đặt **XAMPP** hoặc **MySQL Server**  
4. Cấu hình file `application.properties` cho kết nối cơ sở dữ liệu
